# かずをおぼえよう

子供が数字の読み方を覚えるための、ブラウザだけで動く学習アプリです。
大きく表示した数字にふりがなを付け、桁ごとの意味が見た目で分かるようにしています。

## 機能

- 0 から `9,999,999,999,999,999` までの数字を表示
- 各桁の上に読み方を表示
- `じゅう`、`ひゃく`、`せん` などの位は赤色で表示
- `まん`、`おく`、`ちょう` などの大きな単位はオレンジ色で表示
- 各桁ごとに上向き・下向きのキャレットで数字を変更
- 最上位桁の左側から桁を増減
- スタート / ストップによる自動カウントアップ
- 読み上げモードの ON/OFF
- 読み上げ速度を `とてもゆっくり` / `ゆっくり` / `はやい` / `とてもはやい` から選択
- `よん / し`、`なな / しち`、`きゅう / く`、`ぜろ / れい` の読み方を切り替え

## 使い方

`index.html` をブラウザで開くだけで使えます。

ローカルサーバーで確認する場合:

```sh
python3 -m http.server 8000 --bind 0.0.0.0
```

その後、ブラウザで次を開きます。

```text
http://localhost:8000/
```
