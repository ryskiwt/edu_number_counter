# かずをおぼえよう

子供が数字の読み方を覚えるための、ブラウザだけで動く学習アプリです。
大きく表示した数字にふりがなを付け、桁ごとの意味が見た目で分かるようにしています。

## 機能

- 0 から `9,999,999,999,999` までの数字を表示
- 各桁の上に読み方を表示
- 日本語と英語の2つのレーンを表示し、同時に比べてみることができる
- 日本語は4桁ごと、英語は3桁ごとのまとまりを表現
- 日本語は漢字も表示
- 自動カウントアップに対応し、連続で数字を確認できる
- 日本語と英語での読み上げに対応
- 読み上げ速度を `とてもゆっくり` / `ゆっくり` / `はやい` / `とてもはやい` から選択
- 好みに合わせて複数の読み方に対応（ `よん / し`、`なな / しち`、`きゅう / く`、`ぜろ / れい` ）
